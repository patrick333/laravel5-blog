# laravel5-blog
This is a blog system built with Laravel5.

![Editing in peihuishao.com](/public/images/peihuishao.editing.png)

## Features
* Posts, categories, and tags
* Publish or unpublish
* Markdown editing and syntax highlighting
* Cache
